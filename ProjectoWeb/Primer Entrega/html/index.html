The Best Game Web

¿Que es?

The best Game Web sera un projecto que integre varias tecnologias dentro de una sola paguina web, nuestro reto sera presentar una paguina Web interactiva, con alta calidad visual (atractiva para la vista), con implementacion de almenos 3 juegos dentro de ella.

Prototipo:

En el desarrollo del prototipo inicial tendremos al menos 8 layers principales, dentro de estos estara una pantalla de Login, register, un Home en el momento de iniciar sesion, la galeria de juegos que te dirigira a la pantalla del juego, Un menu dentro de este y paguinas informativas explicando los controles (help) y el completo desarrollo del proyecto (about) & How work?.
